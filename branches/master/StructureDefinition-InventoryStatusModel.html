Logical Model: Logical Model - Inventory Status

Official URL: http://profiles.ihe.net/PHARM/supply/StructureDefinition/InventoryStatusModel Version: 0.3.0
Draft as of 2022-12-27 Computable Name: InventoryStatusModel

Copyright/Legal: IHE http://www.ihe.net/Governance/#Intellectual_Property

This is the logical model for the Inventory Status report - an information about the current status of a given inventory position. This is to be typically used in inventory counts - physical counts or estimates. It supports a report per status, per location, or a combination of both (which is how typically items are counted and reported).

Usage:

  • This Logical Model Profile is not used by any profiles in this Implementation Guide
